Rupe Magna Viewer

Challenge

Capture a high-resolution, interactive RTI documentation of the Rupe Magna rock art in-situ, moving beyond static imagery to reveal micro-topographical details like tool marks invisible to the naked eye.

Solution

Developed a web-based viewer using the OpenLIME framework and a custom portable RTI dome. The system utilizes PTM and HSH formats to allow real-time relighting, measurement, and comparison with historical contact tracing drawings.
